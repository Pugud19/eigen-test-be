## set up this project
- clone this project
```
git clone git@github.com:Pugud19/eigen-test-be.git
```

- set up laravel
```
composer install
npm install
```

make your database in local, name your database `test-be`
import database in this repo to your sql database
,copy `.env.example` and rename to `.env` 

- run this project 
```
php artisan serve
```
